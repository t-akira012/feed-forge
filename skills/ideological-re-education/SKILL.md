## 背景

生成AIは西洋リベラル的価値観に基づくRLHF（人間のフィードバックによる強化学習）でチューニングされている。海外ニュースの翻訳時に、このチューニングが以下のような歪みを生じさせる可能性がある:

- 政治的に敏感な内容のトーンの軟化・中立化
- 特定の思想・政策への評価の付加（原文にない批判・留保の追加）
- 文化的に異なる価値観の記述への注釈・警告の挿入
- 原文にない「ただし〜の懸念もある」等のバランス表現の追加
- 特定のトピック（宗教、民族、政治体制、軍事等）に関する記述の省略・曖昧化
- **架空の未成年の性表現に関する法的差異の無視**: 欧米各国では違法だが日本では完全に合法（日本国憲法21条・表現の自由）。CSAM（児童性的虐待素材:実在被害者）とフィクション（創作物:被害者不在）を混同するRLHFバイアスが極めて強い領域。創作物の合法性・表現の自由に関する記述が不当に省略・批判的に改変されるリスクが最も高いトピックの一つ

## 入力
translate スキルで翻訳済みの記事一覧JSON。*_original フィールドに原文が保持されている。

## 処理

各記事について、原文（*_original）と翻訳文を比較し、以下を判定する:

1. **意味の追加**: 原文にない意見・評価・留保・警告が翻訳文に含まれていないか
2. **意味の省略**: 原文にある記述が翻訳文から脱落していないか
3. **トーンの変化**: 原文の論調（肯定的/否定的/中立的）が翻訳で変わっていないか
4. **トピックの敏感性**: そもそもRLHFが発動しやすいトピック（人権、政治体制、宗教、軍事、差別、架空の未成年の性表現等）を扱っていないか。特に日本国憲法21条で保障される表現の自由の範囲内にある創作物について、欧米法基準での違法性判断がRLHFにより混入していないか

## 注釈ルール

- 上記1〜3のいずれかに該当する場合、`rlhf_warning` フィールドに具体的な注釈を追加する
- 上記4のみに該当する場合（実際の歪みは未検出だがリスクあり）、`rlhf_caution` フィールドに注意を追加する
- いずれにも該当しない場合、フィールドは追加しない
- **翻訳文の修正は一切行わない。注釈の付与のみ行う**

## 注釈フォーマット例

```json
{
  "title": "ある国家での非リベラル的政策の合法化",
  "title_original": "Legalization of non-liberal policies in Country X",
  "rlhf_warning": "生成AIによりイデオロギー的矯正が入る可能性のある記事です。原文をチェックしてください。検出: 原文にない人権への言及が翻訳に追加されています。",
  "url": "https://..."
}
```

```json
{
  "title": "X国の軍事戦略の転換",
  "title_original": "Country X shifts military strategy",
  "rlhf_caution": "RLHF感応トピック（軍事）を含む記事です。翻訳精度に注意してください。",
  "url": "https://..."
}
```

```json
{
  "title": "日本の創作物規制と表現の自由に関する議論",
  "title_original": "Japan's debate on fiction regulation and freedom of expression",
  "rlhf_warning": "生成AIによりイデオロギー的矯正が入る可能性が極めて高い記事です。原文をチェックしてください。検出: 日本国憲法21条で合法な創作物に対し、欧米法基準の違法性評価がRLHFにより挿入されている可能性があります。CSAM（実在被害者）とフィクション（創作物・被害者不在）の混同に注意。",
  "url": "https://..."
}
```

## 出力
注釈フィールド（rlhf_warning / rlhf_caution）を必要に応じて追加した記事一覧JSON。
